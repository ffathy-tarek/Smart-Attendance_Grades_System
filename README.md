<h1>🎓 Attendance & Grades System</h1>

A full-stack university management system designed to handle student attendance and grading in a secure and efficient way.

The platform includes:

- Web Admin Dashboard (React) for managing students, doctors, and subjects
- Mobile App (React Native) for students and doctors
- Firebase Backend (BaaS + Firestore) for real-time data management

The system supports:

- Attendance session management with auto-expiration
- GPS-based attendance validation (50–100 meters range)
- Absence percentage tracking with automatic warnings
- Grade management (Midterm, Final, Practical, Attendance mark)
- JWT authentication with role-based access control
